# cembtnui

> Made with create-react-library

## Install

```bash
npm install --save cembtnui
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'cembtnui'
import 'cembtnui/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
